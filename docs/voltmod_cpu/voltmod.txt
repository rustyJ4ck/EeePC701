Вольтмод для 701/900 Celeron M 353 ULV:

С завода на моем экземпляре Vcore=0.84V (по спецификации VID=0.94V)
Чтобы поднять напряжение, нужно добавить в цепь обратной связи резистор Rfb (VRM поднимет Vcore для компенсации)
Rfb = 47K -> Vcore += 0.072V (8.5%)
Rfb = 68K -> Vcore += 0.050V (5.9%)
Чем ниже номинал резистора Rfb -> больше компенсация -> выше Vcore.

Также из-за большого порога Vsense (100mV выше относительно Vcore), потребуется добавить подтягивающий резистор в цепь Vsen.
(иначе VRM скорее всего будет уходить в защиту при попытке нагрузить процессор при разгоне).

Rsense = 47K -> Vsen -= 0.068V
Чем ниже номинал резистора Rsense -> Vsense ниже -> стремится к Vcore

--
Пример: Желаемая конфигурация:
FSB=110Мгц Частота=990Мгц Vcore=0.893V
-> Rfb=68K Rsense=47K
-> Vcore прибавит 50mV Vsense уменьшится на 68mV

На схеме VRM красным показано куда нужно добавить два резистора
